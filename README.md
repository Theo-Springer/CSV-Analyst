# CSV Analyst

Un outil d'exploration de données léger et interactif pour transformer vos fichiers CSV bruts en analyses visuelles exploitables en quelques secondes.

## 📌 À propos du projet
L'idée de **CSV Analyst** est de combler le fossé entre un tableur Excel rigide et un script de data science complexe. Ce dashboard permet à n'importe quel utilisateur de charger un jeu de données, d'en comprendre la structure statistique et de générer des visualisations sans écrire une seule ligne de code.

Ce projet m'a permis de mettre en pratique la manipulation de données avec **Pandas** et la création d'interfaces dynamiques en **Python**.

## ✨ Fonctionnalités clés
* **Analyse Automatique** : Calcul instantané des indicateurs clés (moyenne, médiane, min/max, écart-type).
* **Diagnostic de Données** : Détection des valeurs manquantes et aperçu des types de colonnes.
* **Visualisation Interactive** : Création de graphiques dynamiques (Histogrammes, nuages de points) avec Plotly.
* **Exploration Fluide** : Filtrage par colonne pour isoler les variables importantes.

## 🛠️ Stack Technique
* **Backend & Logique** : Python 3
* **Traitement de données** : Pandas
* **Interface Utilisateur** : Streamlit
* **Graphiques** : Plotly Express
